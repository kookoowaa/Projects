## 설치 전 참조사항

- 남는 HDD, SSD를 활용할 수 있다면 best, 아니라면 파티션을 나눠서 듀얼부팅 가능
- 남는 HDD, SSD 설치 시 SATA 슬롯이 없다면 SATA-E 슬롯을 활용 가능 
- SATA-E 슬롯은 SATA 슬롯 2개+@로 구성되어 있으며, 일반 SATA 슬롯 2개로도 활용 가능
